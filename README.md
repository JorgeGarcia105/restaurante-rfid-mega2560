# restaurante-rfid-mega2560

Proyecto: Sistema de toma de pedidos para restaurante (Arduino Mega 2560).

Resumen rápido
- Firmware para Arduino Mega 2560: `Proyecto_cocina.ino`.
- Interfaces: LCD 16x2, teclado 4x4, RFID (MFRC522), display 7‑seg y botones de cocina.
